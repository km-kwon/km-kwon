## 🚀 What I'm Focusing On

단순히 화면을 그리는 것을 넘어, **대용량 데이터 환경에서도 쾌적한 렌더링**을 유지하는 프론트엔드 아키텍처에 관심이 많습니다.

- ⚡ **고성능 데이터 시각화:** 초당 80,000건 이상의 스트리밍 로그를 UI 멈춤(Freezing) 없이 처리하는 렌더링 최적화
- 🖥️ **데스크톱 애플리케이션:** React와 Electron을 결합한 산업용 분석 도구 및 사내 툴 개발
- 🏎️ **자율주행 시뮬레이션 연동:** CARLA 기반 환경에서 AI 궤적 예측 모델의 실시간 시각화 파이프라인 구축
